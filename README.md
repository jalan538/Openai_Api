# 🤖 Chatbot & Text Summarizer

This project builds a simple **chatbot using API keys** and a **text summarizer** using Python.

---

## 📌 Features

* Chatbot using API
* Secure API key handling
* Text summarization
* Easy to use and modify

---

## 🔑 API Setup

1. Get your API key
2. Add it as an environment variable:

```bash
export API_KEY="your_api_key"
```
